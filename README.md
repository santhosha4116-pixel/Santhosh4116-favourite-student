 Favourite Students App

A simple React.js application that allows users to manage their favourite students using **React Router** and **Context API**.

Features

* 📋 View a list of students on the **Student List** page.
* ❤️ Add students to the **Favourite Students** page with a single click.
* 🚫 The **Add to Favourites** button is automatically disabled after a student has been added, preventing duplicate entries.
* 🔄 Global state management using **React Context API**.
* 🧭 Smooth navigation between pages using **React Router**.
* 📱 Clean, responsive, and reusable component-based design.

## 🛠️ Built With

* React.js
* React Router
* Context API
* JavaScript (ES6+)
* Tailwind CSS

## 📂 Project Structure

* **Navbar** – Navigation between pages.
* **Card** – Displays student information.
* **Favourite Student Card** – Displays students added to favourites.
* **Favourite Container** – Holds and renders all favourite student cards.
* **Student List Page** – Shows all available students.
* **Favourite Students Page** – Displays all favourited students.

## 🎯 How It Works

1. Browse the list of students.
2. Click the **Add to Favourites** button.
3. The selected student is added to the **Favourite Students** page.
4. Once added, the button becomes **disabled** to prevent adding the same student again.
5. Navigate between pages using the navbar.
